Java Web——11-Json_Ajax

Posted on 2020-07-31 | In Java Web

1.Json

1.定义

  1. JSON是一种轻量级的数据交换格式，不仅易于阅读和编写，而且易于机器解析和生成。
  • 轻量级：跟xml相比较
  • 数据交换：客户端和服务器之间业务数据的传递格式

2.JSON在JavaScript中的使用

定义

Json是由键值对组成，并且由大括号包围。每个键由引号引起来，键和值之间使用冒号进行分隔，多个键值对之间用逗号进行分隔。

访问

json本身是一个对象，其中的key就是对象中的一个属性。

常用方法

  1. json的存在形式：
  • 对象形式————json对象————操作json中的数据时使用
  • 字符串形式————json字符串————客户端和服务器进行数据交换时使用
  1. JSON.stringify()————把json对象转换成json字符串
  2. JSON.parse()————把json字符串转换成json对象

2.Ajax请求

1.定义

  1. Ajax是一种创建交互式网页应用的网页开发技术，一种浏览器通过js异步发起请求，局部更新页面的技术。
  • 局部更新，浏览器地址栏不会发生变化；
  • 局部更新，不会舍弃原来页面的内容。

2.jQuery中的Ajax请求

$.ajax方法

属性 说明
url 表示请求的地址
type 表示请求的类型（GET或POST）
data 表示发送给服务器的数据，格式1:name=value&name=value；格式2:{key:value}
success 请求成功响应的回调函数
dataType 响应的数据类型，text-纯文本、xml-xml数据、json-json对象

$get和$post方法

属性 说明
url 请求的url地址
data 发送的数据
callback 成功的回调函数
type 返回的数据类型

$getJSON方法

属性 说明
url 请求的url地址
data 发送给服务器的数据
callback 成功的回调函数

serialize()方法

  1. serialize()方法可以将表单中所有表单项的内容都获取到并拼接成name=value&name=value形式。
